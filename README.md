## Getting Started

To get started, we must install the following:

- [Node.js installation](https://nodejs.org/en/) - must install nodejs to get nextjs.
- [Adding Next.js and React](https://nextjs.org/docs/getting-started) - nextjs setup.
- [Running Tailwind CSS](https://tailwindcss.com/docs/installation) - adding tailwind directives to css.

After everything is installed, we can now create a nextjs app.

  ```npx create-next-app next-website```
  
Open it in your desired editor, in my case, I used [Visual Studio Code](https://code.visualstudio.com/Download). Now, everything is set up. 

## Creating Website

In order to develop this website, we must have the basic knowledge of HTML5 and CSS3. Throughout the project we used our knowledge of flexboxes to create cards, margin and padding to properly display texts and images, and tailwind animations for hover effects. 

In addition, we had to import:

- [React icons](https://react-icons.github.io/react-icons/), to give our links proper icons.
- [Animate on Scroll](https://michalsnik.github.io/aos/), to create a fading animation.
